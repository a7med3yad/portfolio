Breast Cancer Analysis App

← Back to Home

This project is an interactive application designed to analyze breast cancer datasets. Using machine learning algorithms, it predicts cancer malignancy and provides users with easy-to-understand visualizations. The app is built using Streamlit, enabling a smooth and engaging user experience.

Technologies used: Python, Streamlit, Scikit-learn, Pandas
